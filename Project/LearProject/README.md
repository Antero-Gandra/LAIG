# PLOG

## Setup

1. Clone the repository
2. Get a prolog compiler. Suggested: [YAP](https://www.dcc.fc.up.pt/~vsc/Yap/)
3. Consult lear.pl file
4. Execute `start.`
5. Play

## Game Options

- [X] Human vs Human
- [X] Human vs Robot Easy
- [X] Human vs Robot Hard (IA)
- [X] Robot vs Robot Simulation
